# Changelog

## v0.1.0

First public HACS release.

Highlights:

- Local TCP-to-RS485 communication
- Waveshare RS485 to PoE ETH (B) support
- RGB control
- White color temperature support (PTxxx)
- 15 light programs
- Brightness control
- Restore Last State
- Startup Sequence
- Lamp Wake-Up
- Lampen-Sync (PsS)
- Diagnostics
- Home Assistant Power Controller
- Full Home Assistant integration
